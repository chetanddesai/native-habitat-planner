Contribute

Suggest a Plant

Know a native plant that should be included? Open a GitHub issue with the scientific name, region, and why it belongs.

Suggest a Region

Want to see your city represented? Open a GitHub issue with the city/state and either an iNaturalist place ID or bounding box coordinates.

Submit a Pull Request

Developers can fork the repo and use the add-plant or add-city skills to contribute new data. See the README for details.
